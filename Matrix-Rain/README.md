# Matrix Rain Effect 

A console-based Matrix Rain animation created using C.  
This program simulates the famous Matrix digital rain effect using random characters, spacing logic, and real-time rendering.

## Features
- Random character generation
- Controlled probability for realistic rain gaps
- ANSI escape codes for green text color
- Adjustable animation speed
- Continuous real-time terminal output
